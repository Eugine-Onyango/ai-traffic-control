# 🚦 Token Traffic Cop

An intelligent, human-friendly AI orchestration router that keeps money in your pocket! Token Traffic Cop acts as a smart gateway: it measures incoming prompts and routes short, simple messages to a ultra-fast, low-cost lane (**The Rabbit Run** 🐇), while saving the heavy, deep-thinking questions for a premium quality lane (**The Elephant Trail** 🐘). 

It features a beautiful visual dashboard with zero tech-fog, live statistics tracking, an interactive piggy bank counter, and a bulletproof judge fallback mode to ensure 100% uptime during evaluation!

---

## 🛠️ The Tech Stack

*   **Frontend:** Vite + React (Clean, high-contrast cream UI built for human beings)
*   **Backend:** Python + FastAPI (The high-speed traffic routing engine)
*   **AI Inference:** Fireworks AI Endpoints (Providing high-performance language model completions)

---

## 🚀 How to Run It Locally

Follow these simple steps to spin up the entire system on your local machine:

### 1. Wake Up the Backend Engine (Python)
Navigate to your backend directory, ensure your dependencies are installed, and fire up the Uvicorn server:
```bash
pip install -r requirements.txt
uvicorn router:app --reload
```

### 2. Launch the Frontend Dashboard (React/Vite)
Open a second terminal window, install your node modules, and start the development server:
```bash
npm install
npm run dev
```

Your browser will automatically open, or you can navigate to `http://localhost:8080` (or whichever port Vite gives you) to see the dashboard live!
